# Robotics

Object dection and robot trajectory using YOLOv3, Opencv and Ai2thor

## Getting Started

You can get this project using 
```
git clone: https://github.com/pxthanh98/robotics.git
```
### Requirements

pip requirements:
```
ai2thor
keyboard
numpy
matplotlib
opencv
pillow
pynout
numpy
```

### Installing


```
pip install ai2thor
```
```
pip install opencv
```
```
pip install matplotlib
```
```
pip install pynput
```
